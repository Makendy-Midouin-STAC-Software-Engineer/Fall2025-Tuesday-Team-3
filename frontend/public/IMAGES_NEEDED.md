# Required Images for SafeEatsNYC

Place the following images in this `public` folder. They will be automatically served by Vite.

## Home Page Images

1. **placeholder-nyc-skyline.jpg**
   - Hero image for the home page
   - Recommended: 1920x500px or larger
   - Should show NYC skyline (Manhattan skyline, Brooklyn Bridge, etc.)

2. **placeholder-nyc-restaurant.jpg**
   - Feature card image for "Real Inspection Data"
   - Recommended: 400x200px
   - Should show a NYC restaurant exterior or interior

3. **placeholder-nyc-street.jpg**
   - Feature card image for "Search by Borough"
   - Recommended: 400x200px
   - Should show NYC street scene (street signs, neighborhoods, etc.)

4. **placeholder-nyc-food.jpg**
   - Feature card image for "Cuisine Filters"
   - Recommended: 400x200px
   - Should show NYC food scene (food trucks, restaurant windows, etc.)

## App Page Images

5. **Empty State** (reuses placeholder-nyc-skyline.jpg)
   - Empty state image when no search results
   - Uses the same skyline image from the hero section

6. **placeholder-restaurant.jpg.jpg**
   - Default restaurant card image
   - Recommended: 400x200px
   - Should show a generic restaurant/storefront

## Notes

- All images should be optimized for web (JPEG format recommended)
- Use NYC-themed imagery to maintain the authentic feel
- Replace these placeholders with actual images before final deployment
- Images are referenced with absolute paths (e.g., `/placeholder-nyc-skyline.jpg`)
